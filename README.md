## 概述
GoCQ-E，允许用户使用易语言进行可扩展、易于使用、便于学习且可读性高的SDK，应用在社交辅助、功能提升，以及提供了多线程处理耗时操作的可能性。

## 战略
采用了EPL编写主体架构，原生TCP实现完整协议内容，可实现基于云的、多线程、高并发的开发布局。

## 插件编写
CQfunction_中提供了完整可用的编写介绍，并添加了基础的操作模板，允许用户基于GOCQHTTP文档进行扩展开发。
